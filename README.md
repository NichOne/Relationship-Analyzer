## Overview

The **Relationship Analyzer** is a **privacy-first toolkit** for analyzing relationships through **power dynamics, emotional manipulation, and social control**.

It reveals covert manipulation tactics including:

* Intermittent reinforcement
* Trauma bonding
* Weaponized intimacy
* Social credit and reputation control

All data is stored **locally in your browser**, ensuring **full privacy and data sovereignty**.

## Features

### Analysis Modules

* **Social Debt Tracker:** Maps obligations, favors, and guilt leverage
* **Emotional Bait Detector:** Detects love-bombing cycles and trauma bonding
* **Network Power Mapper:** Visualizes control and hierarchy in social networks
* **Boundary Scanner:** Measures violations, gaslighting, and autonomy breaches
* **Status Decoder:** Identifies transactional relationships and virtue signaling
* **Intimacy Weaponization Alert:** Detects exploitation of vulnerabilities
* **Social Credit System:** Monitors reputation-based pressure and conformity enforcement

### Metrics & Scoring

* **Co-dependency Index:** Emotional manipulation & autonomy loss
* **Emotional Leverage Score:** Social debt + intimacy weaponization
* **Boundary Integrity Rating:** Tracks respect for personal boundaries
* **Predictive Risk Score:** Forecasts escalation using trends & current scores

### Platform Features

* Drag-and-drop visual network mapping
* Historical snapshots & trend analysis
* Side-by-side relationship comparison
* Tagging & notes with critical incident highlighting
* Quick-add action buttons, undo/redo
* Customizable dashboards & widgets
* CSV export & mobile-responsive interface
* AI-powered analysis (Together.ai, OpenRouter, Local LLM)

## Usage

1. Open the **Overview Dashboard** to see metrics across relationships.
2. Use **modules** to analyze individuals or groups.
3. Tag relationships with patterns (gaslighting, manipulation, genuine).
4. Add notes; key incidents are auto-highlighted.
5. Compare multiple relationships using the **Compare tab**.
6. Monitor **Notifications** for high-risk patterns.
7. Save **snapshots** to track changes over time.
8. Export CSV for external analysis.
9. Apply AI analysis to text or structured data.

> ⚠️ **Disclaimer:** This tool identifies patterns and dynamics; it is **not a psychological diagnostic tool**. For serious concerns, consult a professional.

## Methodology

* **Scoring System:**

  * 0–24: Healthy / Safe
  * 25–49: Caution
  * 50–74: Warning
  * 75–100: Critical

* **Trend Analysis:** Forecasts relationship escalation or improvement.

* **Predictive Risk Modeling:** Combines metrics with historical trends to provide early warnings.

## Technical Architecture

* **Platform Independent:** Works with Together.ai, OpenRouter, Local LLM.
* **Local Storage:** All data stored in-browser (localStorage).
* **Export & Backup:** CSV export recommended; optionally fork on GitHub.
* **Data Sovereignty:** No external servers; user retains full control.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/relationship-analyzer.git
cd relationship-analyzer
```

2. Open `index.html` in a modern browser.
3. Configure your LLM provider API key in Settings.
4. Start adding relationships and analyzing patterns.
